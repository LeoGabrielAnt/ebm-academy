EBM Academy V2 Funcional

Esta versão corrige a base de testes anterior usando os módulos completos enviados e um JavaScript consolidado.

Inclui:
- Login visual de teste
- Home principal com ferramentas, documentos e acesso ao treinamento
- Área de treinamento com sidebar dos módulos
- Módulos 01 a 06 completos preservados
- Módulos 07 a 12 como placeholders
- Quizzes randômicos 01 a 06 com alternativas embaralhadas
- Checklist liberador por módulo
- Certificado por módulo
- Modo claro/escuro
- Página de documentos e estrutura para manuais

Como testar:
1. Extraia o ZIP.
2. Abra a pasta no VS Code.
3. Execute index.html com Live Server.
4. Clique em Entrar na EBM Academy.
5. Acesse Iniciar Treinamento.
6. Teste os módulos 01 a 06: checklist > quiz > enviar > refazer > concluir.

Observação:
A tela de login é apenas visual/local. Não há autenticação real nesta versão.
